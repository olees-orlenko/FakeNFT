# Декомпозиция эпика Профиль

## Модуль 1:

#### Верстка
- Экран профиля: аватар, имя, описание, ссылка на сайт, кнопка редактирования (est: 3 часа; fact: x часов).
- Ячейка списка профиля: "Мои NFT" + счетчик и стрелка (est: 1.5 часа; fact: x часов).
- Ячейка списка профиля: "Избранные NFT" + счетчик и стрелка (est: 1.5 часа; fact: x часов).
- Экран "Мои NFT": ячейка NFT (иконка, название, автор, цена, рейтинг) (est: 3.5 часа; fact: x часов).
- Экран "Избранные NFT": карточка NFT в сетке + иконка сердца (est: 3.5 часа; fact: x часов).
- Пустые состояния: "Нет NFT", "Нет избранных" (est: 1 час; fact: x часов).

## Модуль 2:

#### Логика
- Навигация: переходы с профиля в "Мои NFT"/"Избранные NFT" и назад (est: 1.5 часа; fact: x часов).
- Открытие сайта пользователя в WebView (est: 1 час; fact: x часов).
- Сортировка "Мои NFT": выбор критерия и обновление списка (est: 2.5 часа; fact: x часов).
- Сохранение сортировки "Мои NFT" в UserDefaults/@AppStorage (est: 1 час; fact: x часов).
- Удаление из избранного и обновление сетки (est: 2 часа; fact: x часов).
- Обработка состояний загрузки/ошибки для экранов профиля/списков (est: 2.5 часа; fact: x часов).

## Модуль 3:

#### Работа с сетью
- Запрос профиля пользователя (GET) и маппинг модели (est: 2 часа; fact: x часов).
- Запрос "Мои NFT" (GET) и маппинг модели (est: 2 часа; fact: x часов).
- Запрос "Избранные NFT" (GET) и маппинг модели (est: 2 часа; fact: x часов).
- Обновление профиля (PATCH/PUT) при редактировании (est: 2.5 часа; fact: x часов).
- Обновление избранного (POST/DELETE) при клике на сердце (est: 2 часа; fact: x часов).
